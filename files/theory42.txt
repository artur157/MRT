<p>
              Правила головоломки “Ханойская башня” таковы: имеется доска с тремя колышками. На первом из них нанизано несколько дисков убывающего диаметра (самый большой находится внизу). Требуется расположить диски в том же порядке на третьем колышке, причём диски разрешается перекладывать только по одному, а класть больший диск на меньший нельзя. Один колышек используется в качестве вспомогательного. Создадим рекурсивную процедуру, печатающую всю последовательность перекладываний для заданного количества дисков. Алгоритм решения головоломки следующий:</p>
        <ol>
            <li>Переместить верхние n-1 дисков на второй колышек.</li>
            <li>Нижний диск с первого колышка переместить на третий.</li>
            <li>Переместить n-1 дисков на третий колышек, используя первый в
качестве вспомогательного.</li>
            <li>Повторять, пока на третьем колышке не будет сформирована новая пирамида.</li>
        </ol>
            <p><pre>
procedure Hanoi(n, a, b, c: integer);
begin
  if n > 1 then
  begin
    Hanoi(n-1, a, c, b);
    writeln(a, ' -> ', b);
    Hanoi(n-1, c, b, a);
  end else
    writeln(a, ' -> ', b);
  end;</pre></p><p>
	В данной процедуре n – количество дисков, a, b, c – номера штырьков, причём перекладывание производится со штырька a на штырек b при вспомогательном штырьке c.

            </p>
